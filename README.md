Cloud Safe adalah aplikasi web edukatif interaktif yang dirancang untuk menjelaskan konsep dasar keamanan data di cloud storage kepada pengguna awam. Melalui pendekatan *scrolling storytelling* dan simulasi langsung, pengguna diajak memahami bagaimana file mereka dienkripsi dan mengapa menjaga password akun itu sangat penting.

![Cloud Safe Screenshot](https://via.placeholder.com/800x400?text=Cloud+Safe+Preview)

## ğŸ¯ Tujuan Project

Aplikasi ini bertujuan untuk:
1.  **Meningkatkan Kesadaran:** Memberikan pemahaman bahwa data di cloud itu aman karena enkripsi, bukan sihir.
2.  **Edukasi Praktis:** Menunjukkan alur teknis (Upload -> Enkripsi -> Dekripsi) melalui simulasi visual.
3.  **Keamanan Personal:** Menekankan peran pengguna dalam menjaga "kunci brankas" (password) mereka sendiri melalui fitur cek kekuatan password.

## âœ¨ Fitur Utama

*   **Scrolling Storytelling:** Alur cerita yang memandu pengguna langkah demi langkah dengan animasi yang halus.
*   **Simulasi Enkripsi:** Fitur *drag-and-drop* file untuk melihat visualisasi proses hashing dan enkripsi data.
*   **Password Strength Meter:** Alat interaktif untuk mengecek seberapa kuat password pengguna (hanya simulasi, data tidak dikirim ke server).
*   **Kuis Interaktif:** Menguji pemahaman pengguna setelah membaca materi.
*   **Dark Mode:** Dukungan penuh untuk mode gelap dan terang.
